 Quiz Spark is an interactive mobile quiz application developed to provide users with an engaging and enjoyable way to test their knowledge.
 The app presents multiple-choice questions, tracks user scores, and displays results instantly, creating a smooth and user-friendly learning
 experience.

Features:
Interactive multiple-choice quiz questions
Instant answer validation and feedback
Real-time score tracking
User-frienly and responsive interface
Progress tracking throughout the quiz
Final result and performance summary
Restart quiz functionality

Technologies Used:
Flutter – Cross-platform mobile app development
Dart – Application logic and functionality
Material Design – Modern and responsive UI components
VS Code – Development environment

Steps to Run the Project:
1.Clone the repository
2.Navigate to the project directory
3.Install dependencies
4.Run the application

Usage:
Launch the application.
Click on the Start Quiz button.
Read each question carefully.
Select the correct answer from the available options.
Move to the next question.
View your final score after completing the quiz.
Restart the quiz to try again and improve your score.
